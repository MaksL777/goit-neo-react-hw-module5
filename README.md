# Movie Search App 🎬

A React application for searching movies, viewing daily trending films, and exploring detailed movie information, including cast and user reviews. Built using the TMDB API.

## 🚀 Live Demo
[Link to your Vercel deployment](https://your-vercel-link-here.vercel.app)

## 🛠 Technologies Used
* **React** (Hooks, Suspense, Lazy loading)
* **React Router DOM** (Routing, nested routes, location state)
* **Vite** (Build tool)
* **Axios** (HTTP requests)
* **CSS Modules** (Styling)
